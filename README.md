# Vote-app

> Simple chart app which is using MERN stack - MongoDB, Express, React  and Node.js.

### How to start the project:
* <b>Clone the repo </b>
* <b>Navigate to server folder </b>
* <b>npm install </b>
* <b>npm start </b>
* <b>Navigate to client folder </b>
* <b>npm install </b>
* <b>npm start </b>
* <b>Enjoy </b>
